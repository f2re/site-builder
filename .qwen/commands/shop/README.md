# Slash-команды для разработки магазина

## /shop:cart

**Описание:** Инициализировать разработку функционала корзины.

**Что создаётся:**
- Задачи для backend-agent: Redis + Lua скрипты
- Задачи для frontend-agent: UI корзины
- Задачи для testing-agent: тесты race conditions

---

## /shop:checkout

**Описание:** Инициализировать разработку оформления заказа.

**Зависимости:**
- `/shop:cart` — готовая корзина
- `/shop:delivery` — расчёт СДЭК
- `/shop:payment` — интеграция ЮKassa

---

## /shop:delivery

**Описание:** Интеграция со СДЭК v2 API.

**Делегирует:** cdek-agent

**Что реализуется:**
- Расчёт стоимости доставки
- Выбор ПВЗ на карте
- Оформление заказа в СДЭК

---

## /shop:payment

**Описание:** Интеграция с ЮKassa.

**Делегирует:** cdek-agent

**Что реализуется:**
- Создание платежа
- Webhook обработчик
- Обновление статуса заказа

---

## /shop:catalog

**Описание:** Разработка каталога товаров.

**Что создаётся:**
- Backend: CRUD товаров, категории, фильтры
- Frontend: карточки товаров, листинг, фильтры
- Meilisearch: индексация товаров

---

## /shop:orders

**Описание:** Управление заказами.

**Что создаётся:**
- Backend: OrderService, статусы FSM
- Frontend: история заказов, статус заказа
- Admin: список заказов, экспорт CSV
