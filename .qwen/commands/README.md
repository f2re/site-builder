# Slash-команды QWEN

Доступные команды для работы с мультиагентной системой.

## Общие команды

| Команда | Описание |
|---------|----------|
| `/agents:plan <задача>` | Декомпозиция задачи на подзадачи для агентов |
| `/agents:status` | Показать статус текущих задач и агентов |
| `/agents:report <task_id>` | Получить отчёт по выполненной задаче |
| `/agents:validate` | Запустить финальную валидацию проекта (Gatekeeper) |
| `/agents:contract <domain>` | Показать контракты для домена |

## Инициализация фаз

| Команда | Описание |
|---------|----------|
| `/shop:init` | Инициализировать новую фазу разработки магазина |
| `/blog:init` | Инициализировать новую фазу разработки блога |
| `/iot:init` | Инициализировать новую фазу разработки IoT-дашборда |
| `/admin:init` | Инициализировать новую фазу разработки админ-панели |

## Команды магазина

| Команда | Описание |
|---------|----------|
| `/shop:cart` | Инициализировать разработку функционала корзины |
| `/shop:checkout` | Инициализировать разработку оформления заказа |
| `/shop:delivery` | Интеграция со СДЭК v2 API |
| `/shop:payment` | Интеграция с ЮKassa |
| `/shop:catalog` | Разработка каталога товаров |
| `/shop:orders` | Управление заказами |

---

## Примеры использования

### Декомпозиция задачи
```
/agents:plan Добавить функционал корзины с резервированием товаров через Redis Lua
```

### Проверка статуса
```
/agents:status
```

### Валидация перед коммитом
```
/agents:validate
```

### Инициализация фазы
```
/shop:cart
```
